# installing gsap-skills for codex / cursor / other agents

this repo is a set of skill prompt packages. there is no server to run.

## install

1) clone

```bash
git clone https://github.com/Microck/gsap-skills.git ~/gsap-skills
```

2) point your agent tooling at the `skills/` directory, or copy individual skill folders into your existing skill search path.

if you're using opencode specifically, follow `.opencode/INSTALL.md`.

## verify

ask your agent:

> load the `gsap-scrolltrigger` skill and write a pinned horizontal scroll section example
