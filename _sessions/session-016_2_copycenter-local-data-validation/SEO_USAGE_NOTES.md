# SEO USAGE NOTES

## Recomendações Estratégicas
* **H1 / Meta Title:** Usar “Gráfica em Uberlândia” para reforçar o SEO local.
* **Bloco Local / Footer:** Usar “Centro de Uberlândia” na microcopy descritiva para otimização de proximidade, além do NAP estrito.
* **About:** Usar a informação “desde 1978”, desde que haja aprovação prévia do cliente, para gerar autoridade e confiança local.
* **Testimonials / Prova Social:** Utilizar a métrica agregada “mais de 550 avaliações no Google” no título ou subtítulo, como prova social forte (agregada) em vez de focar apenas em 3 textos isolados de clientes.
* **Footer NAP:** Publicar o Name, Address, Phone (NAP) no Footer completo.
* **CTAs:** Usar o telefone/WhatsApp nos botões apenas quando formos atualizar ativamente os `hrefs` (fase posterior).
* **Schema:** Guardar as propriedades validadas para gerar o `LocalBusiness` JSON-LD (Schema) que será injetado na tag `<head>` no fechamento do projeto (Sessão SEO Head/Schema).

## Restrições Atuais (O que NÃO usar ainda)
* **Reviews Individuais:** Não inserir avaliações isoladas/textos de clientes sem que a cópia exata do comentário seja enviada e validada.
* **CNPJ:** Não incluir, visto que não foi confirmado.
* **Horário de Funcionamento:** Não estabelecer promessa fixa sem validação (ainda mais no footer) para evitar desencontros se a gráfica alterar a jornada.
* **Superlativos:** Evitar o uso de "melhor gráfica" ou outros adjetivos não comprováveis objetivamente. Focar em autoridade factual (4,5 estrelas, desde 1978).
